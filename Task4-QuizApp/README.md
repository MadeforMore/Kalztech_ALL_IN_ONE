# Advanced Quiz App

A sophisticated quiz application with dynamic question loading, multiple categories, and advanced features.

## üöÄ Features

### Core Features
- **Dynamic Question Loading**: Fetches questions from Open Trivia Database API
- **Multiple Categories**: General Knowledge, Computer Science, Mathematics, Sports, Geography, History, Animals, Science & Nature
- **Difficulty Levels**: Easy, Medium, Hard, or Mixed
- **Customizable Settings**: Choose number of questions (5-20) and time per question (15-60s or no timer)
- **Real-time Scoring**: Live score tracking with streak counter
- **Progress Tracking**: Visual progress bar and question counter

### Advanced Features
- **Hint System**: Remove incorrect options (50/50 lifeline)
- **Skip Questions**: Skip difficult questions
- **Performance Analytics**: Detailed performance analysis with statistics
- **Answer Review**: Review all answers after quiz completion
- **Share Results**: Share your score on social media or copy to clipboard
- **Keyboard Shortcuts**: Full keyboard navigation support
- **Responsive Design**: Works perfectly on desktop and mobile devices

### User Experience
- **Loading Animations**: Smooth loading screens and transitions
- **Visual Feedback**: Color-coded answers and animated feedback
- **Timer Warnings**: Visual alerts when time is running low
- **Auto-progression**: Automatic question advancement after selection
- **Offline Fallback**: Backup questions when API is unavailable

## üéÆ How to Play

1. **Setup**: Choose your preferred category, difficulty, question count, and timer settings
